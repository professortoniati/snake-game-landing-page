# Snake Evolution

## Objetivo

Criar uma landing page moderna e responsiva para divulgação do clássico Snake Game.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages
- Inteligência Artificial para geração das imagens

## Funcionalidades

- Menu responsivo
- Rolagem suave
- Modal para ampliação de imagens
- Contador animado
- Botão voltar ao topo

## Estrutura

```bash
index.html
css/style.css
js/script.js
images/
```
